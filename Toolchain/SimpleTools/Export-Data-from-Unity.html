Exporting Data from Unity

The Unity template allows you to export gameobjects from Unity to Gama from the Unity Editor.

Using this tool requires to follow 4 steps:

  1. First step: open the model « Plugins models/LinkToUnity/Models/Utilities/ImportGemetriesFromUnity.gaml ».

  1. Second step: run the experiment LoadGeometriesFromUnity.

  2. Third step: In Unity, "GAMA menu -> Export Geometries to GAMA", define the parameter and click on Export Geometries

  1. Fourth step: click « Ok » on the pop-up that appears and immediately add the player to the middleware. If you are too slow to add the player, it will be disconnected and the data will not be exported.
